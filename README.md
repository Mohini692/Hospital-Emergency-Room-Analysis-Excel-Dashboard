# Hospital-Emergency-Room-Analysis-Excel-Dashboard
### 📌 Project Overview

This project analyzes Hospital Emergency Room (ER) data to understand patient flow, waiting time, admission trends, and patient satisfaction.
The goal of this project is to identify operational insights that can help hospitals improve patient experience and resource management.

The analysis was performed using data analysis and visualization techniques to create an interactive dashboard.

### 📊 Dashboard Objectives

The dashboard answers key business questions such as:

1. How many patients visit the emergency room?
2. What is the average patient wait time?
3. What percentage of patients are admitted vs not admitted?
4. How satisfied are patients with the service?
5. Which departments receive the most referrals?
6. What are the patient demographics (age, gender, race)?

## 📂 Dataset Description

The dataset contains records of patients visiting the hospital emergency room.  
It includes patient demographics, admission details, waiting time, and satisfaction scores which help analyze hospital performance and patient experience.

| Column Name | Description |
|-------------|-------------|
| Patient Id | Unique identifier assigned to each patient visit. |
| Patient Admission Date | The date when the patient visited the emergency room. |
| Patient Gender | Gender of the patient (Male / Female). |
| Patient Age | Age of the patient at the time of the hospital visit. |
| Patient Race | Race category of the patient recorded by the hospital. |
| Department Referral | The hospital department where the patient was referred for further treatment. |
| Patient Admission Flag | Indicates whether the patient was admitted to the hospital (Admitted / Not Admitted). |
| Patient Waittime | Time (in minutes) the patient waited before receiving medical attention. |
| Patient Satisfaction Score | Rating provided by the patient based on their experience in the emergency room. |
